# 🌙 EaveVPN Feeds

**  Репозиторий с автоматически обновляемыми VPN-конфигурациями.**

## 📌 О проекте

Этот репозиторий содержит обработанные списки прокси-серверов (VLESS, VMess, Trojan и др.) 
с автоматическим обновлением каждые 1–2 часа.

⚙️ Возможности

- 🔄 **Автообновление** — через GitHub Actions по расписанию
- 🌍 **200+ стран** — автоматическое определение по флагу/названию
- 📝 **Форматирование** — единый стиль имён для всех ссылок
- ⚪⚫ **White/Black List** — разделение по типам списков
- 📊 **Статистика** — заголовок с датой, временем и количеством

🎯 Для кого

Могут использовать все!  

📁 Структура
├── vless-white.txt # Белый список VLESS
├── vless-black.txt # Чёрный список VLESS
├── vmess-mixed.txt # Смешанные VMess
├── process.py # Скрипт обработки
└── .github/workflows/ # Автоматизация
